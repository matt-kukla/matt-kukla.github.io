Information

MVL is an OCaml library for manipulating and evaluating expressions in numerous multivalued propositional logic systems.

Documentation

Module-specific documentation pages:
  • bnp - Belnap four-valued logic.
  • tern - Kleene/Łukasiewicz ternary logic.

Development

Source code is available on GitHub.
