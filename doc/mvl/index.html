Information

MVL is an OCaml library providing implementations of several common multivalued logic systems.

Documentation

Module-specific documentation pages:
  • Mvl.Bnp - Belnap four-valued logic.
  • Mvl.Tern - Łukasiewicz ternary logic.

Build and Installation

Clone the repository from GitHub. MVL uses dune as its build system. Run dune build to compile, and dune install to install.

Development

Source code is available on GitHub.
