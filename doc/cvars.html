Functions

corbernseq(n, p, delta)

Description:

Generate sequence of correlated Bernoulli variables. Uses Monte Carlo simulation of a binomial-correlated random graph.

Standard:

Fortran 90

Arguments:

n Sequence length of type INTEGER.
p Dependency probability of type DOUBLE.
delta Dependency coefficient of type DOUBLE.
