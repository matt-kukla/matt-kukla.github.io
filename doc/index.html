This is a list of various libraries and applications I have written, along with the associated documentation (when it exists). Source code is available on GitHub.

All of my work is free software, usually licensed under the GNU GPL v3 or some variation thereof.

  • arxpdf - shell script for downloading arXiv preprints as PDFs.
  • ltl - a linear temporal logic system.
  • MKPOV - library of POV-Ray textures, macros, other addons.
  • pyantenna - antenna theory tools in Python.
  • tern - Kleene and Łukasiewicz ternary propositional logic.
  • tofn - typed ordered fuzzy numbers (TOFNs) in OCaml.
