arxpdf

A shell script for downloading arXiv preprints as PDFs.

Prerequisites

  • curl

Usage

  • arxpdf [arXiv id]... to download a PDF of [arXiv id]

  • arxpdf [arXiv id]vn... to download the nth version of [arXiv id]

This script accepts both the old arXiv identifier format (subject.SC/YYMMNNN) and the current identifier format (YYMM.number).

Options

  • -h, --help — Display help

  • -v,--version — Display version number
