A list of various libraries and applications I have written, with associated documentation. Source code is available on GitHub.

All of my work is free software, licensed under the GNU GPL v3 or some variation thereof.

  • arxpdf - download arXiv preprints as PDFs from your shell.
  • cvars - generate sequences of correlated Bernoulli variables in FORTRAN.
  • ltl - linear temporal logic.
  • mkpov - library of POV-Ray textures, macros, other addons.
  • mvl - multivalued logic in OCaml.
  • pyantenna - antenna theory tools in Python.
  • tofn - typed ordered fuzzy numbers (TOFNs) in OCaml.
